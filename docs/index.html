Sushi - SSH Client

SSH access for Linux hosts, plus optional Gemini voice help.

Sushi - SSH Client provides reliable SSH access to your Linux hosts, including Raspberry Pi OS instances. Connect with password or key-based authentication, run commands, and keep session logs. Optional Gemini assistance can translate natural requests into safe shell commands and summarize the results for you.

GitHub repo Privacy policy

Core features

  • SSH access to Linux hosts (password or key authentication).
  • Session output and local logs.
  • Optional Google Drive log uploads.

Gemini assistance

  • Generate command suggestions from natural requests.
  • Summarize results after commands run.
  • Optional voice input for Gemini requests.

Notes

  • No app-specific account required.
  • Gemini features use your API key.
  • Voice input uses your device speech service.
